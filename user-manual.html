User Manual

1. Getting Started

Registration and Login

Registration

Registration Form: Create your account with email and security features

  1. Click on the "Register" tab in the welcome screen
  2. Fill in the required information:
    • Email address
    • Password (minimum 6 characters)
    • Name
    • Gender selection
    • Security question (for account recovery)
    • Security answer
  3. Click "Register" to create your account

Login

Login Screen: Access your account with email and password

  1. If you have a registered account:
    • Enter your email
    • Enter your password
    • Click "Login"
  2. Forgot Password?
    • Click "Forgot password?" link
    • Enter your email address
    • Answer your security question
    • Create a new password

3. Active Session Management

Active Participants List

Session Information View

Session Information

  • View detailed session information:
    • Session ID
    • Room location
    • Start time
    • Current duration
    • Running cost
  • Monitor participant activity:
    • See who's currently in the session
    • Track individual join times
    • View participant count

Host Controls

  • Session host has additional controls:
    • End the entire session
    • Transfer host privileges to another participant
    • Manage participant access
  • Host Transfer Process:
    • When the host attempts to leave the session:
      1. A prompt will ask if they want to transfer host responsibilities
      2. If they choose yes, a modal will appear showing all active participants
      3. Click on a participant's name to select them as the new host
      4. The transfer happens automatically, and the original host leaves the session
    • Alternative options when leaving as host:
      • End the entire session instead of transferring
      • Leave without transferring (if confirmed)

‚ö†Ô∏è Session Controls

  • Use "Leave Session" to exit without ending the session
  • Only the current host can use "End Session"
  • Host privileges can be transferred to another participant
  • If you're the last participant and host, you'll be asked to end the session
  • Empty sessions without a host will remain active until manually ended
